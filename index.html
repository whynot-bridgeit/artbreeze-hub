아트브리즈 공간 대관 이용 약관

제1조 (목적) 본 약관은 명동 아트브리즈의 공간 대관 서비스 이용에 관한 제반 사항을 규정함을 목적으로 합니다.

제2조 (신청 및 승인) 대관 신청은 온라인 신청서 작성으로 접수됩니다. 시설 운영팀이 검토 후 3영업일 내 승인 여부를 이메일로 통보합니다.

제3조 (대관료 및 결제) 대관료는 공간별·시간대별로 별도 산정됩니다. 중구 주민·직장인은 최대 40% 할인 혜택이 적용됩니다.

제4조 (취소 및 환불) 대관 7일 전 취소: 전액 환불 / 3~6일 전: 50% 환불 / 2일 이내: 환불 불가

제5조 (시설 이용 의무) 이용자는 시설 및 장비를 선량한 관리자로서 이용하여야 하며, 이용자 귀책으로 발생한 시설 훼손은 배상 책임을 집니다.

제6조 (개인정보) 수집된 개인정보는 대관 서비스 제공 목적으로만 사용되며, 관련 법령에 따라 보호됩니다.

① Google Sheets → 확장 프로그램 → Apps Script → 아래 코드 붙여넣기
function doPost(e) { const sheet = SpreadsheetApp .getActiveSpreadsheet().getSheetByName("대관신청"); const data = JSON.parse(e.postData.contents); // 첫 행 헤더 자동 생성 if (sheet.getLastRow() === 0) { sheet.appendRow(["ID","접수일","상태","이름","연락처", "이메일","공간","시작일","종료일","목적","상세"]); } sheet.appendRow([ data.id, data.submittedAt, "대기", data.name, data.phone, data.email, data.spaceName, data.dateStart, data.dateEnd, data.purposes.join("/"), data.detail ]); // 관리자 알림 이메일 (선택) MailApp.sendEmail({ to: "[email protected]", subject: "[대관신청] " + data.name + " · " + data.spaceName, body: "일정: " + data.dateStart + " " + data.timeStart + " ~ " + data.timeEnd + "\n상세: " + data.detail }); return ContentService .createTextOutput(JSON.stringify({ok:true})) .setMimeType(ContentService.MimeType.JSON); }
② 배포 → 새 배포 → 웹앱 → "모든 사용자" 액세스 → URL 복사 후 아래 코드의 URL 교체
// HTML 파일 맨 아래 <script> 안에 추가 const SHEETS_URL = "https://script.google.com/macros/s/YOUR_ID/exec"; async function sendToGoogleSheets(app) { try { const res = await fetch(SHEETS_URL, { method: "POST", mode: "no-cors", // CORS 우회 body: JSON.stringify(app), headers: { "Content-Type": "application/json" } }); console.log("시트 저장 완료"); } catch(err) { console.error(err); } } // submitForm() 안에서 saveApps() 호출 직후 추가: // sendToGoogleSheets(app);

신청서마다 Notion 데이터베이스에 페이지가 자동 생성됩니다. 칸반 보드로 상태 관리, 팀 협업이 가능합니다. notion.so/my-integrations에서 Integration Token 발급 후 사용합니다.
Notion Integration Token + Database ID 설정
const NOTION_TOKEN = "secret_xxxxxxxxxxxxxxxx"; const DATABASE_ID = "xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx"; async function saveToNotion(app) { await fetch("https://api.notion.com/v1/pages", { method: "POST", headers: { "Authorization": "Bearer " + NOTION_TOKEN, "Notion-Version": "2022-06-28", "Content-Type": "application/json" }, body: JSON.stringify({ parent: { database_id: DATABASE_ID }, properties: { "신청ID": { title: [{ text: { content: app.id } }] }, "이름": { rich_text: [{ text: { content: app.name } }] }, "공간": { select: { name: app.spaceName } }, "시작일": { date: { start: app.dateStart } }, "연락처": { phone_number: app.phone }, "상태": { select: { name: "대기" } } } }) }); } // submitForm() 안에서: saveToNotion(app);

supabase.com에서 프로젝트를 생성하면 PostgreSQL DB를 REST API로 바로 사용할 수 있습니다. 실시간 업데이트, Row Level Security, 관리자 대시보드가 내장되어 있습니다.
① Supabase에서 테이블 생성 (SQL Editor)
create table rental_applications ( id text primary key, submitted_at timestamptz default now(), status text default '대기', name text not null, phone text, email text, org text, space_name text, space_floor text, date_start date, date_end date, time_start time, time_end time, purposes text[], detail text, headcount int, type text, budget text, equipment text ); -- 외부 접근 허용 (anon key 사용 시) alter table rental_applications enable row level security; create policy "insert_public" on rental_applications for insert with check (true);

const SUPABASE_URL = "https://xxxx.supabase.co"; const SUPABASE_KEY = "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXV..."; async function saveToSupabase(app) { const res = await fetch( SUPABASE_URL + "/rest/v1/rental_applications", { method: "POST", headers: { "apikey": SUPABASE_KEY, "Authorization": "Bearer " + SUPABASE_KEY, "Content-Type": "application/json" }, body: JSON.stringify({ id: app.id, name: app.name, phone: app.phone, email: app.email, space_name: app.spaceName, date_start: app.dateStart, date_end: app.dateEnd, purposes: app.purposes, detail: app.detail }) } ); if (res.ok) console.log("Supabase 저장 완료"); } // submitForm() 안에서: saveToSupabase(app);
